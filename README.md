# CodeAlpha_MusicPlayer
A simple and responsive music player built using HTML,CSS and JavaScript with play ,pause, next and previous song functionality.For the CodeAlpha frontend Development InternShip.
## Features
- play and pause songs
- Next and previous song controls
- Song title display
- User-friendly interface
- Responsive design
## Technologies Used
- HTML
- CSS
- JavaScript
## Project Structure
CodeAlpha_MusicPlayer/
- index.html
- song1.m3
- song2.mp3
## How to run
1. Download or clone the repository.
2. Make sure 'song1.mp3' and 'song2.mp3' are in the project folder.
3. Open 'index.html' in your browser or run it using live server in VS code.
